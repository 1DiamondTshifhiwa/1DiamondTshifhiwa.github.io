OVERVIEW OF THE THING

Brief overview of IoT

Brief overview of IoT The internet of Things, or IOT, refers to the vast number of devices that previously lacked internet connectivity but in recent times are now connected to the internet.
These devices are all collecting and sharing data, either to each other or to server computers that will process and extract certain information from the data.
Thanks to the arrival of affordable but yet computationally capable microchips it is possible to turn everything ranging from devices as small as medicinal pills to contraptions as big as aeroplanes.
In recent times we have seen newly connected IoT devices such as smart fridges, toasters, lamps being connected to the internet powering a whole new concept of smart houses the internet of things is making the fabric of the world around us greatly smarter and much more responsive, merging the digital and physical universe.

Description of the IOT project

For this practical project we are required to contribute to the IoT, by creating a webpage or web application that will be able to interface with a ‘Thing’ that is connected to the internet. This thing is a plant that is equipped with 3 sensors measuring the humidity, temperature, and ambient light of plant. These sensors are further connected to an Arduino that further sends this data to an online IoT database called things speak. The plant is further equipped with an LED panel with a resolution of 4x4 LED lights. The interface of this web page or web application should be able to receive the environmental data from the device using an API provided by the IoT database and it should also be able to manipulate the colours of the individual pixels. In summary the main idea of this project is that we should be able to acquire information and be able to manipulate the device or “thing” without being in direct content with it, through the internet hence the internet of things

Inputs and Outputs of the smart object

Inputs and Outputs of the smart object The device is equipped with sensors that provide environmental information as output such as the plant humidity, temperature, and ambient light.
The device can also accept input though the internet in the form of an API that can manipulate the colours the individual LEDs in the LED matrix. Smart object user interface
The interface of this IoT project is a website that will be implemented using various web application technologies.
This website will provide charts of the plant environmental information over time and a unique application that will enable a user to manipulate the colours of the individual LEDs in the LED matrix.
